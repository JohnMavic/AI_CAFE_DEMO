Conway’s Game of Life – Sphere Edition

A 3D twist on the classic cellular automaton mapped onto a rotating sphere. The Game of Life was invented in 1970 by British mathematician John Horton Conway (1937–2020), who spent most of his career at the University of Cambridge and later at Princeton University. Conway made groundbreaking contributions to group theory, number theory, coding theory, and combinatorial game theory; his Game of Life became one of the most famous examples of cellular automata and emergent complexity. This interactive Sphere Edition was developed by Martin Hämmerli and features real-time WebGL rendering, multiple preset patterns, adjustable simulation speed, color schemes, and interactive camera controls.

GitHub – Where the World Builds Software

Founded in 2008 by Tom Preston-Werner, Chris Wanstrath, PJ Hyett, and Scott Chacon as an independent Git hosting startup. Acquired by Microsoft in October 2018 for $7.5 billion. Today the largest source-code platform on earth with over 100 million developers and 420+ million repositories. Core features include version control, code review, CI/CD via GitHub Actions, project management, and AI-powered coding through GitHub Copilot.

Git – Distributed Version Control

Created by Linus Torvalds in April 2005 to manage Linux kernel development after the proprietary tool BitKeeper revoked its free license. Free and open-source under GPL-2.0, maintained by Junio Hamano since its inception. Git is now the de-facto standard for version control, used by virtually every software project worldwide and serving as the foundation for GitHub, GitLab, and Bitbucket. Over 100 million developers rely on Git daily.

Visual Studio Code – Code Editor

Free, open-source code editor developed by Microsoft, first released in April 2015. Built on the Electron framework under the technical leadership of Erich Gamma. Quickly became the world’s most popular editor with over 14 million monthly active users. Key features include IntelliSense, an integrated terminal, built-in Git support, a vast extension marketplace with 50,000+ extensions, and deep integration with GitHub Copilot for AI-assisted development.

Markdown Live Preview

Free, browser-based Markdown editor with real-time side-by-side preview. Originally created as an open-source community tool, it provides instant rendering of Markdown syntax without sign-up or installation. Ideal for quickly drafting and previewing README files, documentation, and notes. Markdown itself was invented in 2004 by John Gruber and Aaron Swartz as a lightweight markup language; it is now the standard format for documentation on GitHub, GitLab, Stack Overflow, and countless other platforms.
